# 🤖 AI 기반 회의 어시스턴트 서비스 (MVP)
> **Django & OpenAI API를 활용한 효율적인 회의록 자동 요약 및 핵심 이슈 추출 서비스**

---

## 🎯 1. 프로젝트 개요
본 프로젝트는 반복적이고 비효율적인 회의록 작성 및 공유 프로세스를 개선하기 위해 개발된 **AI 기반 회의 보조 서비스**입니다. 
회의 음성 텍스트(STT) 데이터를 기반으로 인공지능이 핵심 안건, 주요 이슈, 공지사항을 구조화하여 자동 요약함으로써 팀원 간의 싱크(Sync) 맞추는 시간을 획기적으로 단축합니다.

---

## 💡 2. 핵심 기능 및 유저 시나리오
단순히 API를 호출하는 기능을 넘어, 실제 사용자의 업무 흐름을 시뮬레이션하여 **기능의 우선순위를 구조적으로 설계**했습니다.

* **기능 우선순위 설계 원칙**: 사용자 흐름상 **[음성 녹음 및 텍스트화]**가 선행되어야 **[AI 요약본 생성]**이 의미를 가질 수 있다는 논리적 근거에 기반하여, 2주간 핵심 기능(MVP)을 집중 개발 및 완성했습니다.
* **회의록 자동 요약**: 긴 회의 텍스트 원본을 투입하면 GPT-3.5-turbo 모델을 활용해 4가지 핵심 축(회의 제목, 주요 이슈, 새로운 상황, 추가 안건)으로 구조화된 요약을 출력합니다.
* **보안 및 환경변수 관리**: `os.environ.get` 및 서비스 계정 키 격리 관리를 통해 API 인증 정보를 소스코드와 분리하여 안전하게 관리합니다.

---

## 🛠️ 3. 기술 스택 (Tech Stack)

### Backend / Framework
* **Python 3.x**
* **Django 4.x / 5.x**

### AI & Data API
* **OpenAI API** (GPT-3.5-turbo)
* **Google Cloud Speech-to-Text (STT) API**
